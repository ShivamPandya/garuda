## Pre-requisites
- Python 3.x
- PIP 

## Steps to try alpha version:

1. Clone this repo in your system
``` bash
git clone git@github.com:ShivamPandya/garuda.git
```

2. Swith to the cloned directory
``` bash
cd garuda
```

3. Install the package
``` bash
pip install -e .
```

4. Have fun with Garuda ðŸ¦…
